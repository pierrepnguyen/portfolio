Pet Adoption



type :

Web Development

started :

6 February 2019

completed :

9 February 2019

This application was made with AngularJS as the front end and Node and Express as the back end. Users are able to post pets with details and other uses can like or adopt these pets. Users can also update details about the pets. This web application features full CRUD with MongoDB. The project was designed with materialize css library.
